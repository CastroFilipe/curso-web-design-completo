Pseudo-classe

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado.

:hover

Um estilo pode ser aplicado quando o cursor do mouse passa no elemento.

LVHA :link — :visited — :hover — :active

Estilos que podem ser aplicados em links.
:link - Define o estilo a ser aplicado a um link não visitado.
:visited - Define o estilo a ser aplicado a um link já visitado.
:hover - Como mencionado anteriormente, define o estilo ao passar o cursor do mouse no elemento.
:active - Define o estilo aplicado ao link no momento de execução do clique.
