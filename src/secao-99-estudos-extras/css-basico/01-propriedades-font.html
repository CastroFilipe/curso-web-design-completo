color

A propriedade color define a cor da fonte. Os valores válidos para essa Propriedades são os tradicionais valores para aplicação de cores como valores em hexadecimal, Rgb, Rgba etc.

Alguns exemplos:

  • Cor definida em hexadecimal
  • Cor definida com RGB
  • Cor definida com variável fixa: red, blue, orange, pink etc.

font-family

Define a família de fontes que será utilizada na renderização do texto. Alguns exemplos populares são as combinações:

  • Georgia, 'Times New Roman', serif
  • Helvetica, Arial, sans-serif

font-size

A propriedade font-size define o tamanho da fonte, que pode ser expresso nas unidades px, em, rem, etc.

  • font com tamanho 16px
  • Fonte com tamanho de 1.2em (1.2 referente ao tamanho da fonte do elemento pai)
  • fonte com tamanho de 1.2rem (1.2 referente ao tamanho da fonte do elemento raiz(html))

font-style

A propriedade font-style define o estilo da fonte, podendo ser italic, oblique ou normal

  • Texto normal
  • Texto italic
  • Texto oblique

font-variant

A propriedade font-variant está relacionada com textos em CAIXA ALTA e define a forma como os caracteres são apresentados. Os valores possíveis para essa propriedade são “normal” e o “small-caps”

normal: Texto Camel Case De Exemplo

TEXTO EM CAIXA ALTA DE EXEMPLO

small-caps: Texto Camel Case De Exemplo

font-weight

A propriedade define o peso da fonte, que vai de 100 a 900. Para ter disponível todos os valores dessa faixa é necessário que a fonte instalada tenha todos os valores disponíveis. Geralmente as fontes instaladas na máquina do usuário possuem apenas as variações normal(400) e bold(700)
