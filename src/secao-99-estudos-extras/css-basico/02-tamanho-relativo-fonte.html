Unidade em

A unidade em é relativa ao tamanho da fonte(font-size) do elemento pai. O paragrafo abaixo tem como elemento pai a section, que possui font-size: 16px (herdados do body);

Texto de exemplo com font-size = 1.2em = 1.2 x 16px = 19.2px
Elemento span interno ao paragrafo com font-size: 1.2em = 1.2 x 19.2 = 23,04px

Unidade rem

A unidade rem é relativa ao tamanho da fonte(font-size) do elemento raiz da página(html). Em configurações padrões, os navegadores geralemnte possuem 16px de tamanho de fonte.

Texto de exemplo com font-size = 1.2rem = 1.2 x 16px = 19.2px
Elemento span interno ao paragrafo com font-size: 1.2rem. O tamanho de fonte do elemento pai não interfere.

Unidade %

Font-size definido em % é relativa ao tamanho da fonte do elemento pai.

Texto de exemplo com font-size = 150% = 1.5 x 16px = 24px
Elemento span interno com font-size: 120% = 1.2 x 24px = 28.8px
